# CredSafe

An end-to-end Credit Risk Analysis and Loan Default Prediction system built using SQL, XGBoost, SHAP, and Streamlit.

## Tech Stack

- Python
- SQLite
- Pandas
- XGBoost
- SHAP
- Streamlit
